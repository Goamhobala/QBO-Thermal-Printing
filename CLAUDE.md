## Project Overview

Build a web application that integrates with QuickBooks Online API to create invoices and print 80mm thermal receipts for a South African timber business (Timber 4 U CC).

**Tech Stack:**

* ✅ **TypeScript** for type safety
* ✅ **React** for frontend (you're familiar with it)
* ✅ **Node.js + Express** backend (TypeScript)
* ✅ **Existing thermal template** (reuse your HTML template)

---

## Technical Architecture

### Stack

* **Backend:** Node.js + Express + TypeScript
* **Frontend:** React 18 + TypeScript + Vite
* **QuickBooks Integration:** `intuit-oauth` SDK
* **API:** QuickBooks Online REST API v3
* **Build Tool:** Vite (fast, modern)
* **Styling:** Tailwind
* **Deployment:** Local development first, then Railway/Render
